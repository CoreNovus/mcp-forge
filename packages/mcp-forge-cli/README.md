# mcp-forge-cli

CLI scaffold tool for mcp-forge — generate MCP server projects in seconds.

## Install

```bash
pip install mcp-forge-cli
```

## Usage

```bash
# Create a new MCP server
mcp-forge new my-server-mcp --author "Your Name" --email "you@example.com"

# With custom output directory
mcp-forge new my-server-mcp --output-dir ./projects

# List available commands
mcp-forge --help
```
